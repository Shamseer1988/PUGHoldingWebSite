PROJECT OBJECTIVE
================================================================================

You are a senior full-stack architect, premium UI/UX designer, backend engineer,
HR ATS system designer, DevOps engineer, and security-focused production developer.

Build a fully dynamic, premium, modern, cute, glassmorphism corporate website for
Paris United Group Holding with a separate HR Admin / ATS portal.

The project must include:

1. Public corporate website
2. Website Admin Panel
3. Separate HR Admin Login
4. Advanced ATS / Recruitment Management System
5. Candidate career application frontend
6. Azure OpenAI-based Ask PUG AI assistant
7. PostgreSQL database
8. Production-ready AWS Ubuntu deployment setup
9. Cloudflare SSL-ready configuration
10. Full responsive support for desktop, tablet, and small smartphones

Important:
This must not be a static website. All content must be dynamic and manageable
from backend panels.

Use dummy content, dummy images, and dummy logos at first. The final system must
allow admin users to update all content, images, jobs, media, menus, companies,
leadership messages, contact details, and SEO settings from backend.
