# HR ATS User Guide (placeholder)

This guide will be filled in starting Phase 8 (HR dashboard) and
expanded through Phase 16 (advanced search + reports).

Planned scope:

- Logging in at `/hr/login`.
- Reading the HR dashboard (KPIs and Recharts widgets).
- Creating, editing, and closing job openings.
- Reviewing candidate applications from the public careers page.
- Manually uploading CVs (single and bulk ZIP).
- Reviewing CV parsing results and the rule-based score breakdown.
- Reading the optional AI candidate review (advisory only).
- Moving candidates through the pipeline (`CV Received` →
  `Joined` / `Rejected` / `Blacklisted`).
- Scheduling interviews and collecting interviewer feedback.
- Tracking offers and joining status.
- Running advanced searches and exporting reports to Excel / CSV / PDF.
- Managing HR users, roles, and permissions.
- Reading the HR audit log.

Until then the HR route group only contains a placeholder landing
page at `/hr` so the project structure is visible.
